Oval Button Tokens Used

  • --clearco-ovalbutton-height-* - Button heights for each size
  • --clearco-ovalbutton-padding-horizontal-* - Horizontal padding
  • --clearco-ovalbutton-minwidth-* - Minimum widths
  • --clearco-ovalbutton-radius - Border radius (999px for oval shape)
  • --clearco-ovalbutton-shadow-* - Shadow states
  • --clearco-ovalbutton-transition-duration - Animation timing
  • --clearco-ovalbutton-icongap - Space between icon and text
  • --clearco-ovalbutton-loadingopacity - Opacity in loading state

Usage Example

/* Basic oval button */
.oval-button {
  height: var(--clearco-ovalbutton-height-md);
  padding: 0 var(--clearco-ovalbutton-padding-horizontal-md);
  min-width: var(--clearco-ovalbutton-minwidth-md);
  border-radius: var(--clearco-ovalbutton-radius);
  box-shadow: var(--clearco-ovalbutton-shadow-default);
}

/* Size variant */
.oval-button-lg {
  height: var(--clearco-ovalbutton-height-lg);
  padding: 0 var(--clearco-ovalbutton-padding-horizontal-lg);
  min-width: var(--clearco-ovalbutton-minwidth-lg);
}
